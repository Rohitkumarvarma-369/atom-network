## Prerequisites
* Docker and docker-compose are necessary on the provider end.
* [Zrok](https://zrok.io/) is necessary for both consumer and provider.

## Steps to run the project
### Consumer Client
1. `npm install`
2. `npm run dev`
   
### Provider Client
1. `npm install`
2. `npm run dev`

### Onchain Interactions
1. `npm install`
2. `npm run server`
